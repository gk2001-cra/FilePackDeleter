# File Pack Deleter

A simple Java console application that deletes files of a specific type/extension from a given directory.

---

## Features
- Accepts a directory name from the user.
- Displays the total number of files present in the directory.
- Accepts the file extension (e.g., `.txt`, `.log`, `.jpg`) to delete.
- Deletes all matching files from the given directory.
- Provides clear user-friendly messages.

---

## How It Works
1. The user runs the program.
2. The program asks for a directory path.
3. The program checks if the directory exists.
4. The user enters the type of file (like `.txt`).
5. The program deletes all files matching the extension inside the directory.
